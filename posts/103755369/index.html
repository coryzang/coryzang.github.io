(github.com))зҡ„жЎҶжһ¶жқҘеҶҷзҡ„гҖӮеӣ дёәжүӢжҮ’пјҢжүҖд»ҘжІЎжңүз•ҷдёӢе®үиЈ…е’Ңдҝ®ж”№зҡ„и®°еҪ•пјҢд»Ҡе№ҙеӣ дёәиҜ•иЈ…иҪҜд»¶зҡ„ж—¶еҖҷпјҢдёҚж…ҺдёӯжҜ’гҖӮж— еҘҲеҸӘеҘҪйҮҚиЈ…жңәеҷЁпјҢиҖҢеҚҡе®ўд№ҹдёҖе№¶иҚЎз„¶ж— еӯҳдәҶгҖӮйҮҚж–°жҢ‘йҖүд№ӢдёӢпјҢеҶіе®ҡжҚўз”ЁButterflyдё»йўҳеҺ»жӣҙж–°гҖӮ

HexoзҺҜеўғжҗӯе»ә

е®үиЈ…gitе’Ңnode

gitзҡ„е®үиЈ…жҜ”иҫғз®ҖеҚ•пјҢwindowsзі»з»ҹзҡ„иҜқзӣҙжҺҘдёӢиҪҪе№¶е®үиЈ…е°ұеҘҪдәҶгҖӮиҮідәҺnodeзҡ„е®үиЈ…пјҢжңүе…¶д»–зҡ„bloggerиҜҙ:вҖқ**NodeJs 64дҪҚзҡ„зүҲжң¬йңҖеңЁv12.18.0жүҚиғҪзҲ¬еҲ°иұҶз“Јзҡ„ж•°жҚ®гҖӮвҖқ**пјҢжҲ‘йҰ–ж¬Ўи°ғиҜ•зҡ„ж—¶еҖҷпјҢзЎ®е®һеҰӮжӯӨпјҢжүҖд»ҘпјҢиҝҷйҮҢжҲ‘д»¬иҝҳжҳҜйңҖиҰҒе®үиЈ…иҝҷдёӘзүҲжң¬еҗ§гҖӮ
еҸҰеӨ–йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢе®үиЈ…ж—¶е°ҶpathйғҪеҠ е…Ҙе°ұеҘҪдәҶгҖӮ
е®үиЈ…е®ҢжҜ•еҗҺпјҢеҸҜд»ҘжҢүз…§дёӢеӣҫзҡ„е‘Ҫд»ӨиҝӣиЎҢйӘҢиҜҒ

е®үиЈ…Hexo

Hexoе°ұжҳҜжҲ‘д»¬зҡ„дёӘдәәеҚҡе®ўзҪ‘з«ҷзҡ„жЎҶжһ¶пјҢ иҝҷйҮҢйңҖиҰҒиҮӘе·ұеңЁз”өи„‘еёёйҮҢеҲӣе»әдёҖдёӘж–Үд»¶еӨ№пјҢеҸҜд»Ҙе‘ҪеҗҚдёәBlogпјҢHexoжЎҶжһ¶дёҺд»ҘеҗҺдҪ иҮӘе·ұеҸ‘еёғзҡ„зҪ‘йЎөйғҪеңЁиҝҷдёӘж–Үд»¶еӨ№дёӯгҖӮеҲӣе»әеҘҪеҗҺпјҢиҝӣе…Ҙж–Үд»¶еӨ№дёӯпјҢжҢүдҪҸshiftй”®пјҢеҸіеҮ»йј ж ҮзӮ№еҮ»е‘Ҫд»ӨиЎҢ

дҪҝз”Ёnpmе‘Ҫд»Өе®үиЈ…HexoпјҢиҫ“е…Ҙпјҡ

жіЁж„ҸпјҢиҝҷйҮҢзҡ„е‘Ҫд»ӨйғҪжҳҜдҪңз”ЁеңЁеҲҡеҲҡеҲӣе»әзҡ„Blogж–Үд»¶еӨ№дёӯгҖӮ

дёәдәҶжЈҖжөӢжҲ‘д»¬зҡ„зҪ‘з«ҷйӣҸеҪўпјҢеҲҶеҲ«жҢүйЎәеәҸиҫ“е…Ҙд»ҘдёӢдёүжқЎе‘Ҫд»Өпјҡ

иҝҷдәӣе‘Ҫд»ӨеңЁеҗҺйқўдҪңд»Ӣз»ҚпјҢе®ҢжҲҗеҗҺпјҢжү“ејҖжөҸи§ҲеҷЁиҫ“е…Ҙең°еқҖпјҡ

Githubй…ҚзҪ®

GitHubеҲӣе»әдёӘдәәд»“еә“

зҷ»еҪ•еҲ°GitHub,еҰӮжһңжІЎжңүGitHubеёҗеҸ·пјҢдҪҝз”ЁдҪ зҡ„йӮ®з®ұжіЁеҶҢGitHubеёҗеҸ·пјҡBuild software better, together зӮ№еҮ»GitHubдёӯзҡ„New repositoryеҲӣе»әж–°д»“еә“пјҢд»“еә“еҗҚеә”иҜҘдёәпјҡз”ЁжҲ·еҗҚ.http://github.io иҝҷдёӘз”ЁжҲ·еҗҚдҪҝз”ЁдҪ зҡ„GitHubеёҗеҸ·еҗҚз§°д»ЈжӣҝпјҢиҝҷжҳҜеӣәе®ҡеҶҷжі•пјҢжҜ”еҰӮжҲ‘зҡ„д»“еә“еҗҚдёәпјҡ

й…ҚзҪ®д»“еә“е’ҢеҜҶй’Ҙ

еңЁеҚҡе®ўж №зӣ®еҪ•дёӢGit BashпјҢи®ҫзҪ®user.nameе’Ңuser.emailй…ҚзҪ®дҝЎжҒҜпјҡ

з”ҹжҲҗsshеҜҶй’Ҙж–Үд»¶пјҡ

з„¶еҗҺзӣҙжҺҘдёүдёӘеӣһиҪҰеҚіеҸҜпјҢй»ҳи®ӨдёҚйңҖиҰҒи®ҫзҪ®еҜҶз Ғ
з„¶еҗҺжүҫеҲ°з”ҹжҲҗзҡ„.sshзҡ„ж–Үд»¶еӨ№пјҲдёҖиҲ¬жҳҜcзӣҳдёӢз”ЁжҲ·зӣ®еҪ•дёӢпјүдёӯзҡ„id_rsa.pubеҜҶй’ҘпјҢе°ҶеҶ…е®№е…ЁйғЁеӨҚеҲ¶

жү“ејҖGitHub_Settings_keys йЎөйқўпјҢж–°е»әnew SSH Key

Titleдёәж ҮйўҳпјҢд»»ж„ҸеЎ«еҚіеҸҜпјҢе°ҶеҲҡеҲҡеӨҚеҲ¶зҡ„id_rsa.pubеҶ…е®№зІҳиҙҙиҝӣеҺ»пјҢжңҖеҗҺзӮ№еҮ»Add SSH keyгҖӮ
еңЁGit BashдёӯжЈҖжөӢGitHubе…¬й’Ҙи®ҫзҪ®жҳҜеҗҰжҲҗеҠҹпјҢиҫ“е…Ҙ ssh git@github.com пјҡ

еҰӮдёҠеҲҷиҜҙжҳҺжҲҗеҠҹгҖӮиҝҷйҮҢд№ӢжүҖд»Ҙи®ҫзҪ®GitHubеҜҶй’ҘеҺҹеӣ жҳҜпјҢйҖҡиҝҮйқһеҜ№з§°еҠ еҜҶзҡ„е…¬й’ҘдёҺз§Ғй’ҘжқҘе®ҢжҲҗеҠ еҜҶпјҢе…¬й’Ҙж”ҫзҪ®еңЁGitHubдёҠпјҢз§Ғй’Ҙж”ҫзҪ®еңЁиҮӘе·ұзҡ„з”өи„‘йҮҢгҖӮGitHubиҰҒжұӮжҜҸж¬ЎжҺЁйҖҒд»Јз ҒйғҪжҳҜеҗҲжі•з”ЁжҲ·пјҢжүҖд»ҘжҜҸж¬ЎжҺЁйҖҒйғҪйңҖиҰҒиҫ“е…ҘиҙҰеҸ·еҜҶз ҒйӘҢиҜҒжҺЁйҖҒз”ЁжҲ·жҳҜеҗҰжҳҜеҗҲжі•з”ЁжҲ·пјҢдёәдәҶзңҒеҺ»жҜҸж¬Ўиҫ“е…ҘеҜҶз Ғзҡ„жӯҘйӘӨпјҢйҮҮз”ЁдәҶsshпјҢеҪ“дҪ жҺЁйҖҒзҡ„ж—¶еҖҷпјҢgitе°ұдјҡеҢ№й…ҚдҪ зҡ„з§Ғй’Ҙи·ҹGitHubдёҠйқўзҡ„е…¬й’ҘжҳҜеҗҰжҳҜй…ҚеҜ№зҡ„пјҢиӢҘжҳҜеҢ№й…Қе°ұи®ӨдёәдҪ жҳҜеҗҲжі•з”ЁжҲ·пјҢеҲҷе…Ғи®ёжҺЁйҖҒгҖӮиҝҷж ·еҸҜд»ҘдҝқиҜҒжҜҸж¬Ўзҡ„жҺЁйҖҒйғҪжҳҜжӯЈзЎ®еҗҲжі•зҡ„гҖӮ

жҺЁйҖҒзҪ‘з«ҷ

blogж №зӣ®еҪ•йҮҢзҡ„_config.ymlж–Үд»¶з§°дёәз«ҷзӮ№й…ҚзҪ®ж–Үд»¶пјҢдёӢдёҖжӯҘе°ҶжҲ‘д»¬зҡ„HexoдёҺGitHubе…іиҒ”иө·жқҘпјҢжү“ејҖз«ҷзӮ№зҡ„й…ҚзҪ®ж–Үд»¶_config.ymlпјҢзҝ»еҲ°жңҖеҗҺдҝ®ж”№дёәпјҡ

еҸӮиҖғеҰӮдёӢпјҡ иҝҷйҮҢиҰҒжіЁж„Ҹзҡ„жҳҜжҜҸдёӘпјҡеҗҺйқўжңүдёҖдёӘз©әж јгҖӮ

дҝқеӯҳз«ҷзӮ№й…ҚзҪ®ж–Үд»¶гҖӮ

е…¶е®һе°ұжҳҜз»ҷhexo d иҝҷдёӘе‘Ҫд»ӨеҒҡзӣёеә”зҡ„й…ҚзҪ®пјҢи®©hexoзҹҘйҒ“дҪ иҰҒжҠҠblogйғЁзҪІеңЁе“ӘдёӘдҪҚзҪ®пјҢеҫҲжҳҫз„¶пјҢжҲ‘д»¬йғЁзҪІеңЁжҲ‘д»¬GitHubзҡ„д»“еә“йҮҢгҖӮжңҖеҗҺе®үиЈ…GitйғЁзҪІжҸ’д»¶пјҢиҫ“е…Ҙе‘Ҫд»Өпјҡ

иҝҷж—¶пјҢжҲ‘д»¬еҲҶеҲ«иҫ“е…ҘдёүжқЎе‘Ҫд»Өпјҡ

е…¶е®һз¬¬дёүжқЎзҡ„ hexo deploy е°ұжҳҜйғЁзҪІзҪ‘з«ҷе‘Ҫд»ӨпјҢdжҳҜdeployзҡ„зј©еҶҷ,еӣ дёәеҗҺйқўиҰҒй…ҚзҪ®иұҶз“ЈжҸ’д»¶пјҢеҰӮжһңеҶҷжҲҗ hexo d дјҡжңүеҶІзӘҒпјҢжҲ‘д»¬иҝҳжҳҜеҶҷжҲҗ hexo deployгҖӮе®ҢжҲҗеҗҺпјҢжү“ејҖжөҸи§ҲеҷЁпјҢеңЁең°еқҖж Ҹиҫ“е…ҘдҪ зҡ„ж”ҫзҪ®дёӘдәәзҪ‘з«ҷзҡ„д»“еә“и·Ҝеҫ„пјҢеҚі http://xxxx.github.io (зҹҘд№ҺжҺ’зүҲеҸҜиғҪдјҡеҮәзҺ°вҖқhttp://вҖңеӯ—ж ·пјҢеҸӮиҖғдёӢеӣҫ) жҜ”еҰӮжҲ‘зҡ„xxxxе°ұжҳҜжҲ‘зҡ„GitHubз”ЁжҲ·еҗҚпјҡ

дҪ е°ұдјҡеҸ‘зҺ°дҪ зҡ„еҚҡе®ўе·Із»ҸдёҠзәҝдәҶпјҢеҸҜд»ҘеңЁзҪ‘з»ңдёҠиў«и®ҝй—®дәҶгҖӮ

е®үиЈ…еҸҠеә”з”ЁButterflyдё»йўҳ

е®үиЈ…дё»йўҳ

еңЁдҪ зҡ„ Hexo ж №зӣ®еҪ•йҮҢпјҢ

еә”з”Ёдё»йўҳ

дҝ®ж”№ Hexo ж №зӣ®еҪ•дёӢзҡ„ _config.ymlпјҢжҠҠдё»йўҳж”№дёәbutterfly

е®үиЈ…жҸ’д»¶

еҰӮжһңдҪ жІ’жңү pug д»ҘеҸҠ stylus зҡ„жёІжҹ“еҷЁпјҢиҜ·дёӢиҪҪе®үиЈқпјҡ

еҚҮзҙҡе»әи®®

еҚҮзә§е®ҢжҲҗеҗҺпјҢиҜ·еҲ° Github зҡ„ Releases з•Ңйқў жҲ–иҖ… ж–ҮжӘ”дёғ жҹҘзңӢж–°зүҲзҡ„жӣҙж–°е…§е®№гҖӮ

йҮҢйқўжңүж ҮжіЁ _config ж–Үд»¶зҡ„еҸҳжӣҙе…§е®№пјҲеҰӮжңүпјүпјҢиҜ·ж №жҚ®е®һйҷ…жғ…жіҒжӣҙж–°й…ҚзҪ®е…§е®№гҖӮ

дёәдәҶжёӣе°‘еҚҮзҙҡдё»йЎҢеҗҺеёҰдҫҶзҡ„дёҚдҫҝпјҢеңЁ hexo зҡ„ж №зӣ®еҪ•еҲӣе»әдёҖдёӘж–Үд»¶ _config.butterfly.ymlпјҢе№¶жҠҠдё»йўҳзӣ®еҪ•зҡ„ _config.yml е…§е®№еӨҚеҲ¶еҲ° _config.butterfly.yml еҺ»гҖӮ( жіЁж„Ҹ: еӨҚеҲ¶зҡ„жҳҜдё»йЎҢзҡ„ _config.yml ,иҖҢдёҚжҳҜ hexo зҡ„ _config.yml)

жіЁж„Ҹпјҡ дёҚиҰҒжҠҠдё»йўҳзӣ®еҪ•зҡ„ _config.yml еҲ жҺү
жіЁж„Ҹпјҡ д»ҘеҗҺеҸӘйңҖиҰҒеңЁ _config.butterfly.ymlиҝӣиЎҢй…ҚзҪ®е°ұиЎҢгҖӮ
еҰӮжһңдҪҝз”ЁдәҶ _config.butterfly.ymlпјҢ й…ҚзҪ®дё»йЎҢзҡ„ _config.yml е°ҮдёҚжңғжңүж•ҲжһңгҖӮ

HexoжңғиҮӘеӢ•еҗҲдҪөдё»йЎҢдёӯзҡ„_config.ymlе’Ң _config.butterfly.ymlйҮҢзҡ„й…ҚзҪ®пјҢеҰӮжһңеӯҳеңЁеҗҢеҗҚй…ҚзҪ®пјҢдјҡдҪҝз”Ё_config.butterfly.ymlзҡ„й…ҚзҪ®пјҢе…¶дјҳе…ҲеәҰиҫғй«ҳгҖӮ
