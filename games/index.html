那些游戏

游戏的真正成功之处就在于恰到好处地激活了受众体内的快乐因子——内啡肽

泰坦陨落2 Titanfall 2
2022-03-25 / 角色扮演 机甲 FPS 动作 /
心中的高达梦，在这个游戏里面得到了满足。也是我玩过的所有机甲类游戏中的最棒的一个。不管是单兵模式的跳跃、设计的质感、操作感，还是机甲模式下的迟滞感和重火力的压制，都打磨的非常非常好，期待下一作！

这部的劳拉黑化了。更多地是要硬刚敌人，在潜行之外，加上了格斗和狙杀的元素。算是情节不够，打斗凑吧
古墓丽影 崛起 Rise of the Tomb Raider
2022-03-25 / 冒险 动作 解谜 角色扮演 / ★★★★☆ 推荐
第二部的制作相比之下，不如第一部来的惊艳，在操作的难度上提高了，谜题也有提升，但故事的节奏和感情的变化有点不搭调

街头霸王4 Street Fighter IV
2022-03-25 / ACT 格斗 街头霸王 日本 /
当年ipad刚刚推出，我从美国第一时间拿到。特别装了这款游戏，给周围的小哥们震的一愣一愣的，都忘了嘲笑我的菜鸟水平了。
恐龙危机2 ディノクライシス2
2022-03-25 / 童年回忆 Capcom 单机游戏 ACT RPG /
这款游戏是后来很多类似游戏的鼻祖，以今天的角度看，关卡设计以及操作的爽快酣畅的感觉以及出色的画风都是不落后！
重返德军总部 Return to Castle Wolfenstein
2022-03-25 / 童年回忆 第一人称射击 FPS / ★★★★★ 力荐
玩到僵尸关的时候，吓得闭着眼，一通乱打枪。这个游戏，非常经典，当时的3A大作。
剑侠情缘
2022-03-25 / 单机游戏 角色扮演 武侠 RPG 西山居 / ★★★★☆ 推荐
这款游戏先出了demo版本，还是装在几张3.5寸盘上，试玩一下，里面居然有怡红院，可惜试玩版中不让进，为了进“怡红院”，特别买了正版游戏。哈哈。游戏在当时还行，最棒的是游戏的音乐，这个比游戏流传的更广泛。
仙剑奇侠传 仙劍奇俠傳
2022-03-25 / 童年回忆 单机游戏 角色扮演 武侠 RPG / ★★★☆☆ 还行
说实话，我玩仙剑奇侠传的时候比较晚，那时已经出了很多画风、音乐更细腻的新作，所以感觉一般吧，特别讨厌它的那种回合制的机械。
杀戮时刻 Kill Switch
2022-03-25 / 单机游戏 角色扮演 / ★★★★★ 力荐
为了玩这款游戏，特别买了一台电脑。当时特别惊艳，动作、枪械升级等等，现在看，这款游戏的想法依然不落后。 记忆深刻！
金庸群侠传 金庸群俠傳
2022-03-25 / 童年回忆 角色扮演 武侠 单机游戏 智冠 / ★★★★★ 力荐
我是一个不折不扣的金庸迷，玩这个游戏的时候才工作，还没有女友。所以，很多个夜晚，我都在游戏的大地图里面不停的闯关，不停的升级。这个游戏，我玩了无数遍。现在看起来，这款游戏实在有点太幼稚，但那时玩它的感觉就是那么舒服。这大概就是回忆杀吧。
大富翁4
2022-03-25 / 休闲 童年回忆 经典 / ★★★★★ 力荐
至今想着我的沙隆巴斯把女友的金贝贝陷害进监狱时，女友扑在怀里，又捶又气的样子。欢乐！
纪念碑谷2 Monument Valley 2
2022-03-25 / 模拟 益智游戏 / ★★★★★ 力荐
延续了前作的品质，这一版本难度有一点增加，增加了一个联动的角色。其他的还是一样的棒，女儿玩的特别开心。
纪念碑谷 Monument Valley
2022-03-25 / 模拟 益智 休闲 / ★★★★★ 力荐
特别棒的一个游戏，很精巧的设计，很舒缓的画面，很益智的关卡，很唯美的画风，是给女儿挑选的一款游戏，特别棒！
